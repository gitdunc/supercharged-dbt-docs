# Data Glossary

## Objective

Provide a shared business vocabulary so data consumers interpret metrics and entities consistently.

## Recommended Glossary Fields

- term
- business definition
- owner/steward
- source system
- approved synonyms
- deprecated synonyms
- examples

## Mapping to This Repo

- Entity names from AdventureWorks can be mapped to glossary terms.
- Link glossary terms to object metadata via `meta.glossary_terms`.
- Show glossary references in the Data Definition panel for each node.

## Governance Process

- business teams propose terms
- data governance council approves terms
- platform team publishes and versions the glossary
